Enables the block settings context menu on right-click in the WordPress block editor.

== Description ==

Enable Context Menu restores right-click functionality in the WordPress block editor. Right-clicking a block opens the native block settings dropdown (the same menu available in the List View), positioned at the cursor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/enable-context-menu` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Open any post or page in the block editor and right-click a block.

== Frequently Asked Questions ==

= Where does the menu appear? =

The context menu appears at the cursor position when you right-click any block in the editor canvas.

= Does it work with the Site Editor? =

Yes! The context menu works in both the post editor and the Site Editor.

= Does it work with custom post types? =

Yes, it is available in the block editor for all post types.
